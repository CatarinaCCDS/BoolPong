Welcome to Bool Pong! üçª

A game that works  better when you're drunk!

Controls:
ENTER: Start Game
SPACE: Shoot the Ball
R: Restart Game
ESC: Exit Game
Objective:
Hit all the cups to win! Keep shooting until all the cups are gone.
Gameplay Tips:
Aim carefully before shooting!
